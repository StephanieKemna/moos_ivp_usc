To run this simulation (assuming you have installed & compiled everything etc):

In a terminal tab:
$ ./launch.sh <options>

To start vehicle:
- click on the DEPLOY ALL button

You should see the AUV start going to a waypoint pattern.
(You can use the arrow keys to move in the pMarineViewer window, i/o to zoom in/out)

Note; pAcommsHandler comes from goby, make sure to install goby:
1) get dccl: http://libdccl.org/ 
   -- libdccl3-dev
2) get goby: https://github.com/GobySoft/goby/wiki/InstallingGoby 
   -- libgoby2-dev libgoby2 goby2-apps goby2-moos
